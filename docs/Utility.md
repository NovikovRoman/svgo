# Utility

- `Grid`

    ```go
    // svgo
    Grid(x, y, w, h, n int, s ...string)
    // float/svgo
    Grid(x, y, w, h, n float64, s ...string)
    ```

    draws a grid of straight lines starting at x,y, with a width w, and height h, and a size of n.
